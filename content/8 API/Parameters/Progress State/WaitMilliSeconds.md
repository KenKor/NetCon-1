# WaitMilliSeconds

Type of: _integer_
Unique: __

Optional parameter, the default value is '-1'. If  the [[WaitTillInitialized|WaitTillInitialized]] is set, this second parameter will specify the maximum of milliseconds that the call will wait before returning.
